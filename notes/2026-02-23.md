Specificity & Dynamic CSS: 

 Changing an element’s style in javascript:
One way -  write directly to style itself:
EX: node.style.backgroundColor = ‘#FFFF0’;
Very expressive and general, not maintainable or flexible 
Or use the element id by giving a className or classList
This is preferred!
You can link using .className
EX: <div> class = “inCart”>
 Writing into the <style> property :   .inCart { background-color: #EFE63F }
This is super maintainable and flexible not very expressive or general; 



CSS Specificity: 
From most specificity value to least specificity value 
(Style Attribute), (ID), ( Class or pseudo class or attribute), (Elements) 
Ex:    0             1                      1                      3
             ul#nav li.active a 
Which one wins? Style attribute will win 
BUT you can use the !important if you want a certain part to win over a style attribute
