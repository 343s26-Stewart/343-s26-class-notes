* CSS Intro:
    HTML is used for the structure and content
    Style and interactivity is left for CSS and JS

    Syntax - a CSS doc consists of style rules
    A rule has a selector that identifies the HTML element that will be affected
    They are followed by property:value pairs (declaration)

    Values - each declaration must contain a value
    
    Colors - use standard color names, rgb, hex, rgba or hsl

    Units of Measure Values:
        Relative units are based on the value of something else
        Absolute units have real world size
