Check out CSS tricks when researching flexbox.

set an element to a flexbox by setting display element to "flex".
children of flex element will be considered flex children.
the direction of the children can be chosen with "flex-direction"
the "justify-content" attribute is used to determine how the content is displayed.
the "align-items" attribute is used to determine how content is displayed in the other axis.
