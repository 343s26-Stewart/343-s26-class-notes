## Javascript
-Use <script> tag to add javascript to your HTML file. 
-Typically put your script tags at the bottom of the body but still inside the body.

-Variables in JavaScript are dynamically typed, meaning that you do not have
to declare the type of a variable before you use it.
-To declare a variable in JavaScript, use either var, const, or let keywords (use mostly just let and const)
    - const does not permit re-assignment

OUTPUT:
-alert(): Displays the content within a browser-controlled pop-up/modal window

-prompt(): Displays a message and an input field within a modal window

-confirm(): Displays a question in a modal window with ok and cancel buttons

DATA TYPES:
-reference types: usually referred to as objects

-primitive types: non-object, simple types
note: JavaScript lets you use primitive types as they are objects

BUILT-IN OBJECTS
-common ones: Object, Function, Boolean, Error, Number, Math, Date, String, and RegExp

Truthy and Falsy
-Everything in JavaScript has an inherent Boolean value
-All values are truthy except false, null, "", '', 0, NaN, and undefined
